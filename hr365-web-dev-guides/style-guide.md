# 样式规范

## 一、字体

1. 字族（font-family）采用 antd 默认；
2. 字号（font-size）：12px、14px、16px、20px、24px、30px 中选择；
3. 字重（font-weight）
    - 常规 Regular 400（默认）
    - 细体 Light 300
    - 级细 Thin 100
4. 字体颜色（color）：
    - 纯黑 Black #000
    - 标题 Title: rgba(0,0,0,.85)
    - 主要文本 Primary Text: rgba(0,0,0,.65)
    - 次要文本 Secondary Text: rgba(0,0,0,.45)
    - 不可用 Disable Text: rgba(0,0,0,.25)
